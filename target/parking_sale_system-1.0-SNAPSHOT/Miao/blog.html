Project Description

郑州市国际会展中心是郑州市郑东新区的标志性建筑，也是郑东新区的奠基工程，它的建设标志着由世界著名建筑规划大师黑川纪章先生构思并获国际规划大奖的郑东新区正式启动。郑州市市委、市政府对郑州市国际会展中心的建设十分关注，市委书记李克同志明确指示要以一流的规划、一流的设计、一流的工程和一流的队伍，建设成为国际一流的标志性建筑。
